# Remix Ecommerce

Production ecommerce platform built with Remix, React, and TypeScript. Deployed on Vercel.

## Tech Stack

- **Framework**: [Remix](https://remix.run/) (React full-stack)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
npm start
```
